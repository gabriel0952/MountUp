# MountUp PRD（Product Requirements Document）

## 1. 產品概述

**MountUp** 是一款專為台灣山友打造的戶外健行紀錄 APP，定位為個人健行日記。
使用者不需登入即可在本機記錄每次出行；登入後資料自動同步至雲端，方便跨裝置存取。

### 目標用戶
- 台灣健行愛好者（初階至進階山友）
- 習慣用手機記錄行程的戶外玩家
- 喜歡在社群媒體分享健行成果的用戶

---

## 2. 使用情境

| 情境 | 描述 |
|------|------|
| 出發前 | 查看預建路線資料庫，規劃行程；勾選裝備清單確認不漏帶 |
| 健行中 | 開啟即時 GPS 追蹤，地圖顯示目前位置與預載 GPX 路線，隨時拍照打卡 |
| 下山後 | 查看本次統計（距離、爬升、時間）；撰寫筆記；生成 SNS 分享卡片 |
| 長期使用 | 瀏覽歷史紀錄；建立個人裝備資料庫 |

---

## 3. 核心功能

### 3.1 活動記錄

每次健行建立一筆活動，包含：

| 欄位 | 說明 |
|------|------|
| 日期 | 出行日期（可手動修改） |
| 路線名稱 | 自由輸入或從路線資料庫選取 |
| 距離 | 公里，GPS 自動計算或手動輸入 |
| 總爬升 | 公尺，GPS 自動計算或手動輸入 |
| 移動時間 | 計時器自動計算 |
| 狀態 | 進行中 / 已完成 |

### 3.2 GPS 即時追蹤 & 地圖

- 啟動追蹤後，每秒記錄 GPS 座標，繪製即時軌跡
- 地圖底圖：**Mapbox Outdoors**（內建等高線、地形陰影、步道標示，離線 Pack 下載）
- GPX 路線疊加顯示於地圖，來源有兩種：
  - **路線資料庫**：從官方預建路線選取，自動載入對應 GPX
  - **外部匯入**：使用者自行匯入 `.gpx` 檔案（Files / AirDrop）
- 顯示目前速度、已行進距離、爬升

追蹤流程：
```
開始追蹤 → 背景持續定位 → 寫入本地 Hive → 結束追蹤 → 計算統計 → 儲存活動
```

### 3.3 照片 / 筆記

- 每筆活動可附加多張照片（帶 GPS 標記）
- 照片清單依拍攝時間排序，可在地圖上查看拍攝位置
- 文字筆記：自由格式，支援 Markdown 基本語法

### 3.4 路線資料庫

- 官方預建台灣百岳及熱門郊山路線
- 每條路線包含：名稱、難度、距離、爬升、GPX 檔案、封面照片、簡介
- 可依難度、地區、距離篩選
- P1 後期：UGC 用戶投稿路線

### 3.5 SNS 分享卡片

兩種格式：

| 格式 | 用途 | 內容 |
|------|------|------|
| 1:1 正方形 | Instagram 貼文 | 路線縮圖軌跡 + 距離/爬升/時間 + 路線名 |
| 9:16 直式 | Instagram 限動 | 全版地圖截圖 + 數據疊加 + APP Logo |

生成流程：
```
選擇活動 → 選格式 → 預覽卡片 → 存至相簿 / 直接分享
```

### 3.6 裝備清單

- 建立多個裝備清單（如「百岳三日」、「郊山半日」）
- 每個清單包含多個裝備項目，可標記類別（衣物、糧食、急救等）
- 出發前勾選確認模式：打勾記錄已備妥，出發後自動重置
- 可複製已有清單作為範本

## 4. MVP 範圍

### P0（首版上線）
- [ ] 活動記錄（手動建立）
- [ ] GPS 即時追蹤 + 地圖（OpenStreetMap + GPX）
- [ ] 照片附加 + 文字筆記
- [ ] 路線資料庫（官方預建，唯讀）
- [ ] SNS 分享卡片（1:1 & 9:16）
- [ ] 裝備清單管理

- [ ] 帳號登入（Supabase Auth）
- [ ] 雲端同步（活動、照片、裝備清單）
- [ ] 路線 UGC 投稿
- [ ] 天氣快照（出行當天天氣紀錄）

### P2（未來擴充）
- [ ] 好友動態 / 社群功能
- [ ] Apple Watch 配對
- [ ] 進階統計（年度回顧、個人紀錄）
- [ ] 離線地圖下載管理

## 5. 設計原則

- **離線優先**：所有 P0 功能無需網路即可使用
- **簡潔直觀**：健行中單手可操作，減少層級深度
- **在地化**：路線資料以台灣為主，支援繁體中文
- **隱私保護**：本機資料預設不上傳；登入為可選功能

---

## 6. 技術棧

> 詳見 `CLAUDE.md`

- Flutter 3.x / Dart 3.x
- 狀態管理：Riverpod
- 導航：go_router
- 後端（P1）：Supabase（Auth + PostgreSQL + Storage）
- 本地儲存：Hive
- Data class：freezed
- 錯誤處理：fpdart (Either)

---

## 7. 非功能需求

- **離線可用**：P0 全功能離線運作，網路僅用於 P1 同步
- **效能**：地圖滑動 60fps，GPS 追蹤不明顯耗電
- **啟動時間**：冷啟動 < 2 秒
- **儲存**：本機照片壓縮後平均 < 2MB / 張
- **安全**：登入後 token 加密儲存，通訊 HTTPS
